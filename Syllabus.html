2.2.1 Test de composants

Objectifs des tests de composants

 Vérifier si les comportements fonctionnels et non-fonctionnels du composant sont tels qu'ils ont été conçus et spécifiés  Renforcer la confiance dans la qualité du composant  Trouver des défauts dans le composant  Empêcher les défauts de passer à des niveaux de test plus élevés


Voici des exemples de produits d’activités qui peuvent être utilisés comme bases de test pour les tests de composants :  Conception détaillée  Code  Modèle de données  Spécifications des composants
